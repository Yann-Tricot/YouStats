#Ce programme permet de traiter les données des documents csv récuperer de l'api youtube


#utilisation du programme

1) Mettre dans les cvs a traiter dans le dossier output 

2) lancer le programme
	py traitement.py <arg1> <arg2>
	<arg1> -> date de début du traitement	 format : "Y.M.D" ex: "2020.09.09"
	<arg2> -> date de fin du traitement	 format : "Y.M.D" ex: "2020.09.11"

3) retrouver les fichiers traité dans le dossier output


#prerequis 

Il n'y a pas de prérequis
